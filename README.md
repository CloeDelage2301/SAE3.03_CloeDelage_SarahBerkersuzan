# SAé 3.01 : Intégration et Développement de Parcours Utilisateurs

> **Projet :**SAÉ 3.Crea.03 | Concevoir des visualisations de données pour le web et un support animé
> **Groupe :** Cloé et Sarah

---

## Présentation du projet

Nous avions pour objectif de créer un site internet de visualisation de données sur une thématique précise. Nous avons choisi l'alimentation et nous nous sommes inspirés du site qqf.fr.

---

## Technique

Nous avons utilisé le HTML pour la structure de la page, le CSS pour la mise en forme et GSAP et Lottie pour les animations. Chaque section (slide) a été réalisée avec CSS Grid lorsque cela était nécessaire. Nous avons évité le positionnement "absolute", à l'exception de la numérotation des pages. Pour ce point spécifique, nous avons sollicité une IA pour nous aider a comprendre son utilité. Nous avons également conçu une maquette sur Figma pour visualiser le rendu final avant de passer au développement, toute notre démarche est dessus ainsi que l'explication de notre DA. Enfin, nous avons réalisé une brochure synthétisant les grandes parties du projet ainsi que les données clés liées à notre sujet. Toute les illustration on était faite à la main sur Illustrator et notre Htlm a été vérifier avec W3C validator.
